# Discord Intelligence Bot
Your internal observability + analytics console, powered by Discord. 

## 1️⃣ SYSTEM OVERVIEW
What we are building

A Discord bot that:

** Joins servers normally (invite OR member link join)

** Tracks member joins across all servers

** Sends structured event logs to your central operations server

** Keeps logs organized in dedicated channels


## Core Channels (in your central server)

Create these manually:

#joins-feed
#system-logs
#event-reports   (optional — future GPT analysis output)

## 2️⃣ Core System Capabilities (Phase 1: Foundation)

** Guild join detection

** Member join tracking

** Central logging

** Structured message output

** Stable deployment on Render

** Auto-restart & uptime
